# Telegram Bot - Animal Shelter (sky.pro)

Создание телеграм бота для приюта животных
http://t.me/Java9Team3_AnimalShelterBot

Бот позволяет:

- Узнать информацию о приюте.
- Как взять собаку из приюта.
- Прислать отчет о питомце.
- Позвать волонтера.
